# Analog

Analog is a meta-framework for building applications and websites with [Angular](https://angular.io).

## Goals

- Vite-based
- Low config
- File-based routing
- Hybrid SSR/SSG support
- Integrated CLI

## Packages

- @analogjs/vite-plugin-angular: An Angular Plugin for building applications with Vite
